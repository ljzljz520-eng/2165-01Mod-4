# 明日方舟风格迷你塔防（C 终端版）

这是一个用 C 语言实现的简化塔防小游戏，支持本地运行与 Docker 一键启动。

## 🚀 快速启动

### 方式一：本地运行

```bash
make
./arknights_td
```

### 方式二：Docker 运行

```bash
./run-docker.sh
```

或：

```bash
docker compose build
docker compose run --rm -it -e TD_MODE=play game
```

## 游戏目标

- 默认守住 `5` 波敌人即可通关
- 基地生命降到 `0` 则失败
- 战斗阶段为自动战斗，不能输入指令

## 基础规则

- 地图大小固定为 `12x8`
- `#` 为敌人路径，不能部署干员
- 初始基地生命：`10`
- 初始费用：`20`
- 敌人到达终点：基地生命 `-1`
- 击杀敌人：获得费用奖励
- 每波结束：额外 `+5` 费用

## 干员规则

- 部署费用：`8`
- 初始属性：Lv1，射程 `2`，伤害 `3`
- 升级上限：Lv3
- 首次升级费用：`10`
- 后续每次升级费用增加：`+6`
- 每次升级效果：射程 `+1`，伤害 `+2`
- Lv.3 后可专精为精英干员（费用 `20`），获得真实伤害

## 伤害与防御机制

游戏中有三种伤害类型：

| 伤害类型 | 获取方式 | 作用 |
|---------|---------|------|
| 物理伤害 | Lv.1 ~ Lv.2 干员 | 被敌人护甲减免 |
| 法术伤害 | Lv.3 干员（觉醒） | 被敌方法术抗性减免 |
| 真实伤害 | 专精后的精英干员 | 无视所有防御 |

敌人拥有两种防御属性：

- **护甲**：减免物理伤害
- **法抗**：减免法术伤害

减伤规则：

- 每点护甲/法抗提供 8% 减伤
- 最高减伤上限为 60%
- 真实伤害完全无视防御

## 指令说明

- `place x y` / `部署 x y`：部署干员
- `upgrade x y` / `升级 x y`：升级干员
- `promote x y` / `专精 x y`：专精为精英干员（真实伤害）
- `start` / `开始`：开始当前波次
- `map` / `地图`：显示地图
- `log on/off` / `日志 开/关`：切换战斗详细日志
- `help` / `帮助`：显示帮助
- `tips` / `提示`：显示新手提示
- `quit` / `退出`：退出游戏

## 战斗日志

- 战斗日志会显示每次攻击的伤害类型、原始伤害、实际伤害和减伤数值
- 简单难度默认关闭详细日志，避免刷屏
- 可通过 `log` / `日志` 命令手动切换

地图图例：

- `T`：普通干员（物理伤害）
- `A`：高级干员 Lv.3（法术伤害）
- `S`：精英干员（真实伤害，无视防御）
- `E`：敌人
- `#`：路径

## 终端输入编辑

准备阶段输入支持行内编辑（TTY 交互模式）：

- `←` / `→`：按 UTF-8 字符移动光标
- `Home` / `End`：跳到行首/行尾
- `Backspace` / `Delete`：删除光标前/后的 UTF-8 字符
- 支持插入式编辑，输错后可就地修改，无需整行重输

说明：

- 非交互输入（重定向、管道、脚本）会自动回退到标准按行读取，以兼容自动化测试与 Docker demo。

## 建议开局

1. 输入 `地图`
2. 输入 `部署 3 2`
3. 输入 `部署 5 4`
4. 输入 `开始`
5. 波次结束后再根据费用继续部署或升级

## 可选环境变量

- `TD_MAX_WAVES`：总波次（默认 `5`，最大 `30`）
- `TD_TICK_MS`：每个战斗 tick 的延时毫秒（默认 `180`）
- `TD_WAVE_CLEAR_MS`：每波结束停顿毫秒（默认 `700`）
- `TD_NO_CLEAR`：禁用清屏（用于日志/取证，默认关闭）
- `TD_DIFFICULTY`：难度设置（`easy` / `normal` / `hard`，默认 `normal`）
  - 简单难度：默认关闭详细战斗日志
  - 普通/困难难度：默认开启详细战斗日志
